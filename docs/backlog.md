### **Секция 2: Каталог курсов (публичный доступ)**

| ID | Priority | Story | Who | What do | For what | Blocked by |
|----|----------|-------|-----|---------|----------|------------|
| CAT-1 | | Интерактивная навигация по каталогу в шапке сайта | Гость | Может просматривать категории и курсы в них через выпадающее меню | Чтобы быстро находить интересующие курсы, не уходя с текущей страницы | - |
| CAT-2 | | Просмотр страницы со списком курсов категории | Гость | Может видеть список курсов на отдельной странице | Чтобы изучить все предложения в рамках одной темы | CAT-1 |
| CAT-2.1 | | *Фильтрация курсов на странице категории* | Гость | Может фильтровать курсы по сложности | Чтобы быстрее найти курс подходящего уровня | CAT-2 |
| CAT-2.2 | | *Сортировка курсов на странице категории* | Гость | Может сортировать курсы по дате или сложности | Чтобы упорядочить список в удобном для себя виде | CAT-2 |
| CAT-5 | | Просмотр уроков курса | Гость | Может видеть список уроков курса | Чтобы оценить структуру и содержание курса перед началом обучения | CAT-2 |

**CAT-1: Интерактивная навигация по каталогу в шапке сайта**
*   **DoD (Definition of Done):**
    1.  **UI в шапке**: В `header` сайта на публичных страницах отображается горизонтальный список названий категорий.
    2.  **Логика**: В этот список попадают только категории, в которых есть хотя бы один *опубликованный* курс.
    3.  **Hover-эффект**: При наведении курсора на название категории, под ним плавно появляется выпадающий список с курсами.
    4.  **Адаптивность списка**: Высота выпадающего списка адаптируется под высоту экрана пользователя (чтобы избежать выхода за пределы видимой области), при необходимости добавляя внутреннюю прокрутку.
    5.  **Клик-эффект**: Клик по названию категории ведет на страницу полного списка курсов этой категории (реализуется в `CAT-2`).
    6.  **Рендеринг**: Данные о категориях и вложенных в них курсах подготавливаются на стороне сервера и передаются в HTML-шаблон, который рендерится и отдает готовую навигацию как часть публичных страниц.

**CAT-2: Просмотр страницы со списком курсов категории**
*   **DoD (Definition of Done):**
    1.  **UI**: Отображается страница со списком курсов.
    2.  **Элемент списка**: Содержит название и сложность. При наведении — краткое описание.
    3.  **Клик**: Переход на страницу с уроками (`CAT-5`).
    4.  **Пагинация**: Реализована навигация "Вперед/Назад".
    5.  **Данные**: В списке только опубликованные курсы из выбранной категории.
    6.  **Рендеринг**: Endpoint `/categories/{id}` отдает готовую HTML-страницу.

**CAT-2.1: Фильтрация курсов на странице категории**
*   **DoD (Definition of Done):**
    1.  **UI**: На странице `CAT-2` добавлены элементы для фильтрации (чекбоксы со сложностью).
    2.  **Функциональность**: Применение фильтра перезагружает страницу с параметрами в URL (например, `?level=easy,medium`).
    3.  **Результат**: Список курсов на странице отфильтрован в соответствии с выбором.
    4.  **Рендеринг**: Endpoint `/categories/{id}` умеет обрабатывать параметры фильтрации из URL.

**CAT-2.2: Сортировка курсов на странице категории**
*   **DoD (Definition of Done):**
    1.  **UI**: На странице `CAT-2` добавлен выпадающий список для выбора сортировки.
    2.  **Функциональность**: Выбор опции сортировки перезагружает страницу с параметрами в URL (например, `?sort=-created_at`).
    3.  **Результат**: Список курсов на странице отсортирован в соответствии с выбором.
    4.  **Рендеринг**: Endpoint `/categories/{id}` умеет обрабатывать параметры сортировки из URL.
