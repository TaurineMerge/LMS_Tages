Таблицы и хранилища, которые используются приложением.

Приложение использует следующие базы данных и хранилища:

1. knowledge-base-db - для публичного сайта и админ панели
2. perconal-account-db - для личного кабинета
3. testing-db - для сервиса тестинга
4. keycloak-db - своя база данных keycloak, где хранятся данные для авторизации и аутентификации пользователя
