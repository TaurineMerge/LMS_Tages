# Certificates Router

API endpoints для управления сертификатами.

## Endpoints

| Method | Path | Description |
|--------|------|-------------|
| GET | `/api/v1/certificates` | Список сертификатов |
| GET | `/api/v1/certificates/{id}` | Получить сертификат по ID |
| POST | `/api/v1/certificates` | Создать сертификат |
