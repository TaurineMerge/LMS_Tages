# Validation utilities

Пакет `app.clients.validation` содержит общий загрузчик схем и менеджер
валидации контрактов, которые используются клиентами интеграций.

## Модули

- `contract_manager` — высокоуровневый менеджер валидации (`ContractManager`).
- `schema_loader` — асинхронный загрузчик версионированных JSON‑схем (`SchemaLoader`).
