# LMS_Tages

Learning Management System (LMS) - система управления обучением.

## Структура проекта

- **adminPanel** - административная панель для управления курсами, уроками и категориями
- **publicSide** - публичная сторона с фронтендом на Handlebars для отображения курсов
- **personal-account** - личный кабинет студента (Python)
- **testing** - модуль тестирования (Java)
- **init-sql** - SQL скрипты для инициализации базы данных

## Публичная сторона (publicSide)

Добавлен фронтенд для отображения курсов по категориям:

### Особенности
- Handlebars templates для рендеринга HTML
- Адаптивный дизайн с карточками курсов
- Отображение уровня сложности с цветовой индикацией
- Обрезанное описание курсов
- Отображаются только публичные курсы

### Эндпоинты
- `GET /categories/:categoryId/courses` - Страница со списком курсов категории

### Запуск
```bash
cd publicSide
go run cmd/main.go
```

Подробная документация: [publicSide/views/README.md](publicSide/views/README.md)

## База данных

Для настройки базы данных используйте скрипты из папки `init-sql/`:
- `migrate-001.sql` - создание схемы базы данных
- `populate-999.sql` - наполнение тестовыми данными
- `add-public-courses.sql` - установка курсов как публичных

## Docker

Проект использует Docker Compose для оркестрации сервисов:
```bash
docker-compose up -d
```
